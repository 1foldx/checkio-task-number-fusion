On souhaite fusionner les chiffres deux entiers

On vous propose deux entiers. Pour cette tache, vous devez retourner un entier qui contient les chiffres des entiers passés en argument. Pour ce faire vous devez décomposer les deux entiers (par divisions successives), puis composer le résultat en alternant les chiffres des deux entiers. Exemple: pour 1234 et 8765 {1, 2, 3, 4} et {5, 6, 7, 8} sont les chiffes qui composent les entiers donnés, le résultat sera donc 18273645.

Contraintes

  • str à ne pas utiliser
  • list à utiliser comme pile uniquement

Saisie: Un entier.

Sortie: Deux entiers.

Exemple:

fusion(20, 20) == 2200
fusion(99,11) == 9191
fusion(1234, 8765) == 18273645
fusion(9, 0) == 90

Intérêt de la mission Cette mission vous aidera à comprendre comment manipuler des entiers et comment utiliser un système de numération positionnel.

Précondition :
 0 <= X
les deux nombres ont le même nombre de chiffres.
